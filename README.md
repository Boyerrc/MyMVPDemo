# MyMVPDemo
学习mvp实现的小demo，逻辑简单，便于理解，帮助刚接触mvp+rxjava的人。
基于Ultra Pull To Refresh 实现了下拉刷新，并且修复了Upt和viewpager的事件冲突问题，也封装了加载更多，很适合新人使用。
